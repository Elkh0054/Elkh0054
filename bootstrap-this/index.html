Why Bootstrap?

Bootstrap is an easier way for web designers and developers. It's automatically responsive, less complex than css. In a nutshull, Bootstrap is much easy to understand and use. In this page I will introduce the most important three elements of any design which is "Typography", "Images" and "Colors".

Typography

Bootstrap made it easy to designers to format their typography. You can easily change the color. You can add a background for the font, Highlight a text easily,Underline it or even cross it out.

Changing the weight and size of the text is also very handy while using "Bootstrap". The size can be adjusting using the "fs." followed by the number of the text's size for example fs-1, fs-3,fs-5 .

Images

The most challenging topic for web developers is adjusting the images to be repsonsive on different screens, but with bootstrap the images can be easily responsive with the use of the ".img-fluid"

You can also easily align and style images using Bootstrap.

Colors

The easiest way to change the color of text is using bootstarp. You can also change the color of the text and the background in one command There is also an amazing Theme Palette and colors with wide variables of colors.
